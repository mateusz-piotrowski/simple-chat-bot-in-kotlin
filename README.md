# Simple Chat Bot (Kotlin)

## Description

This is the *Simple Chat Bot (Kotlin)* project I made myself.

<p>Here, at the beginning of your programmer’s path, creating a simple console chat bot will do wonders to guide you through the basics of coding. During this journey you will also play some word and number games that you are going to implement all on your own. Pack up and let’s hit the road, my friend!</p>

Features of the bot:
- Greets the user and introduces itself.
- Asks for the user's name.
- Guesses the user's age based on remainders of division by 3, 5, and 7.
- Counts to any given number.
- Tests programming knowledge with a multiple-choice question.

## Motivation

This project was developed as part of the JetBrains Academy (Hyperskill) curriculum to practice basic Kotlin concepts such as standard input/output, variables, mathematical operations, loops, and conditional statements.

## Getting Started

To run the project locally:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open the project in an IDE that supports Kotlin (e.g., IntelliJ IDEA).
3. Run the `SimpleBot.kt` file located in `Simple Chat Bot (Kotlin)/task/src/bot/SimpleBot.kt`.
